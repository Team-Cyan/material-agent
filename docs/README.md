# Docs

This directory contains the repository's durable documentation layers.

## Layout

- `.agents/`: repo-local agent assets and Codex/harness fast paths
- `docs/ai/`: agent-facing knowledge base
- `docs/roadmap.md`: current repository state and next work
- `docs/operations/`: operator workflows and session handoff notes
- `docs/superpowers/`: historical specs and plans generated during earlier agent workflows
- top-level runbooks such as `harness-runbook.md`, `module-map.md`, and benchmark/model notes

## Reading Strategy

- Start with `docs/ai/project-overview.md` for agent work.
- Read `docs/roadmap.md` before broad repo changes.
- Read `docs/operations/session-handoff.md` only when unfinished work or context carry-over matters.
